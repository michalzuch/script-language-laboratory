# TypeScript Project

## Prerequisites

1. Install [pnpm](https://pnpm.io/)
2. Install [bun](https://bun.sh/)

## Running the Application

1. Start the database:

   ```bash
   pnpm start
   ```

2. Start the backend:

   ```bash
   bun run dev
   ```

3. Start the frontend:

   ```bash
   pnpm dev
   ```
